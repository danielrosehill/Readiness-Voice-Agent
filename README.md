# Readiness Voice Agent

> **Status: WIP (Work In Progress)**

An Android app that uses a conversational voice agent to interactively walk users through emergency preparedness checklists. Voice-first companion to the [Israel Emergency Readiness PWA](https://github.com/danielrosehill/Israel-Emergency-Readiness-PWA).

## Concept

During emergencies or preparation, reading a screen isn't always practical. This app lets users complete checklists hands-free via a voice agent that reads items aloud, accepts verbal confirmation, provides contextual details on request, and flags critical items.

## Checklists Supported

- **BRACED Quick Smoke Test** — 2-minute rapid readiness check
- **Master Checklist** — Full 8-section preparedness audit
- **Situational** — Daytime, WFH, After Shelter, Shower, Bedtime, Leaving Home
- **Special** — Shabbat/Hag, HFC App Config, Mamad Inspection, Home Safety, Pantry Staples

All checklist data sourced from the [PWA project](https://github.com/danielrosehill/Israel-Emergency-Readiness-PWA).
